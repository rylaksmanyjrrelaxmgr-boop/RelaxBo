# ريلاكس مانيجر - بوت إدارة القنوات والمجموعات

الإصدار: 19.1.0

## الميزات
- إدارة القنوات والمجموعات
- نشر تلقائي وجدولة
- أمان متقدم (NSFW، كلمات محظورة، وضع بطيء)
- ردود تلقائية (200 رد مدمج)
- نظام مسابقات متكامل
- واجهة ويب متكاملة
- نسخ احتياطي مشفر
- دعم 12 لغة
- وإحالات وتذكيرات ومستويات

## التشغيل
1. تثبيت المكتبات: `pip install -r requirements.txt`
2. إعداد ملف `.env`
3. تشغيل البوت: `python bot.py`
